# 🐔 NBRP Chicken Harvest

> 🔗 **Discord:** https://discord.gg/YOURDISCORD

A fully networked chicken harvesting system for FiveM.

Players can catch chickens in the world and receive egg rewards. All chickens are server-synced, ensuring consistent behavior across all players.

---

## 📸 Overview

- Approach a chicken
- Press **E** to catch it
- Play catch animation
- Receive random egg rewards
- Chicken disappears and respawns after a delay

---

## ✨ Features

- 🌐 **Server-synced chickens** (OneSync supported)
- 🎯 **Interactive catching system** (TextUI + animation)
- 🎁 **Configurable rewards** (min/max eggs)
- 🌍 **Flexible spawn zones** (radius + density)
- 🔄 **Respawn system** with cooldown
- 🧠 **Server-side validation** (prevents exploits)
- 🧹 **Safe cleanup commands** (no global entity wipes)

---

## 📦 Dependencies

Make sure these are installed and started **before** this resource:

- [`ox_lib`](https://github.com/overextended/ox_lib)
- [`ox_inventory`](https://github.com/overextended/ox_inventory)
