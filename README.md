# 🐔 FiveM Chicken Harvest Script (ox_lib / ox_inventory)

> 🔗 **Discord:** https://discord.gg/YOURDISCORD

A lightweight, server-synced chicken harvesting system for FiveM.  
Players can catch chickens in the world and receive egg rewards, with fully configurable spawn zones and respawn timers.


---

## ✨ Features

- 🌐 Server-synced chickens (OneSync supported)
- 🎯 Press **E** to catch chickens
- 🎬 Animation + progress bar interaction
- 🎁 Random egg rewards (configurable)
- 🌍 Configurable spawn zones (radius + count)
- 🔄 Adjustable respawn timer
- 🧠 Server-side validation (anti-exploit)
- 🧹 Safe entity management (no global ped wipes)

---
